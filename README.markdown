# Bad ~~Wolf~~ Dog

This is a personal fork of [Bad Wolf](https://github.com/sjl/badwolf). A color scheme for Vim, originally pieced together by [Steve Losh](http://stevelosh.com/).

## Why?

Currently there are very minimal differences between this and the original. The main "feature" is that Bad Dog will look identical on both Vim and Neovim. Neovim 0.10 changed the default highlight groups which are used as the base of the colorscheme. So this fork restores the original Vim HL groups when using Neovim.

It also comes with an optional lightline colorscheme.
